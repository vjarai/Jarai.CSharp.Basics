Aufgabe: Arrays
===============

Lassen Sie den Anwender die anzahl von Umsatzwerten eingeben.
Erstellen Sie ein Array für alle umsätze: double[] umsatz = new double[anzahl];
Lassen Sie den Anwender alle Umsatzwerte in einer Schleife eingeben.
Geben Sie die Umsatzzahlen in einer zweiten Schleife in umgekehrter Reigenfolge wieder aus.
